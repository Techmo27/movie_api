Documentation Project: myFlix App

By using the myFlix App you are able to receive information on movies, directors, and genres so that you can learn more about movies you've watched or are interested in. Also, you're able to create a profile so you can save data about your favorite movies.

Request HTTP Method URL Response
Return home of myFlix App GET '/' A welcoming text message.
Return a list of all movies GET '/movies' An array of JSON objects containing data about all the movies.
Return data about a single movie by title GET '/movies/:title' A JSON object holding the data about one single movie. It contains information about the title, genre, directors name and film length. It's structured as follows: { title: "Movie1", genre: "Comedy", director: "Tim Burton", film lenght: "120min"}
Return data about a genre by title GET '/movies/genre/:title' A text message describing the particular genre
Return data about a director by name GET '/movies/directors/:name' A text message with information about the director (bio, birth year, death year).
Allow new users to register POST '/users' A JSON object holding data about the user who has been added in the following structure { name: "Sophie Berger", username: "s.berger135", email: "sophieberger@gmail.com", password: "13579sb"}
Allow users to update their user info PUT '/users/:username' A text message indicating which information about the user has been updated
Allow users to add a movie to their list of favorites POST 'users/:username/favorites/:ID' A JSON object holding data about the movie tthat has been added to the list of favourites, like so: { title: "Movie1", genre: "Comedy", director: "Tim Burton", film lenght: "120min"}
Allow users to remove a movie to their list of favorites DELETE 'users/:username/favorites/:ID' A text message indicating whether the movie was successfully removed
Allow existing users to deregister DELETE '/users/:username' A text message indicating whether the users account is successfully deleted
