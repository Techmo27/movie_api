Documentation Project: myFlix App

By using the myFlix App you are able to receive information on movies, directors, and genres so that you can learn more about movies you've watched or are interested in. Also, you're able to create a profile so you can save data about your favorite movies.

Request HTTP Method URL Response Response Schema/ Data Type
Return home of myFlix App GET '/' A welcoming text message. String
Return a list of all movies GET '/movies' An array of JSON objects containing all data about all the movies. [{"Movie1"}, {"Movie2"}...]
Return data about a single movie by title GET '/movies/:title' A JSON object holding the data about one single movie. It contains information about the title, genre, directors name and film length. { Title: {type: String}, Description: {type: String}, Genre: { Name: String, Description: String }, Director: { Name: String, Bio: String }, Actors: [String], ImagePath: String, Featured: Boolean
Return data about a genre by title GET '/genres/:genre' A JSON object describing the particular genre { Name: String, Description: String }
Return data about a director by name GET '/directors/:name' A JSON object with information about the director. { Name: String, Bio: String }
Allow new users to register POST '/users' A JSON object holding data about the user who has been added. { Username: {type: String}, Password: {type: String}, Email: {type: String}, Birthday: Date, FavoriteMovies: [] }
Allow users to update their user info PUT '/users/:username' A JSON object holding data about the user who's data has been updated. { Username: {type: String}, Password: {type: String}, Email: {type: String}, Birthday: Date, FavoriteMovies: [] }
Allow users to add a movie to their list of favorites POST '/users/:username/favorites/:movieID' A JSON object holding data about the user and her/his all favorite movies including the currently added. { Username: {type: String}, Password: {type: String}, Email: {type: String}, Birthday: Date, FavoriteMovies: [{Movie1}, {Movie2}...] }
Allow users to remove a movie to their list of favorites DELETE '/users/:username/favorites/:movieID' A JSON object holding data about the user and her/his all favorite movies excluding the deleted one. { Username: {type: String}, Password: {type: String}, Email: {type: String}, Birthday: Date, FavoriteMovies: [{Movie1}, {Movie2}...] }
Allow existing users to deregister DELETE '/users/:username' A text message indicating the users account is successfully deleted String
